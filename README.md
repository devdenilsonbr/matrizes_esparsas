# Matriz Esparsa

## Como Executar
1. Para iniciar, execute o arquivo `start.sh` conforme o sistema operacional:
- **Windows (Git Bash ou WSL)**:
    ```bash
    sh start.sh
    ```
- **macOS**:
    ```bash
    ./start.sh
    ```
- **Linux**:
    ```bash
    ./start.sh
    ```
